# PyTNL Docs

This is a docs page for the [PyTNL](https://gitlab.com/tnl-project/pytnl) project.

This project uses [MkDocs](https://mkdocs.org) along with the [material](https://squidfunk.github.io/mkdocs-material/) theme.

## Requsities

- Python
- Pip

## Dev Setup

- Clone this repository
- Create virtual environment using `python -m venv .venv`
- Run `pip install -r requirements.txt`
- To Serve run `mkdocs serve` -> Serves the development server on localhost:8000
- To Build run `mkdocs build` -> Builds and outputs static site assets to /public directory
